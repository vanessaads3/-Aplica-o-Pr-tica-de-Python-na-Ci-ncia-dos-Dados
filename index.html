Projeto Python com PyScript

Este trabalho foi desenvolvido com o Google Colab, um ambiente de codificação interativo que permite o uso de gráficos e texto inline.

Descrição da atividade

A ciência de dados tem se consolidado como uma das áreas mais estratégicas para empresas que desejam tomar decisões baseadas em evidências. Nesta atividade, exploramos a aplicação prática da linguagem de programação Python — uma das ferramentas mais versáteis e poderosas para análise de dados — no contexto de um problema real trazido por uma empresa de cartão de crédito.

O objetivo é analisar um conjunto de dados que reúne informações demográficas e financeiras de 30.000 clientes, com o intuito de compreender padrões de inadimplência. Utilizando bibliotecas como pandas, realizamos a importação, inspeção e exploração dos dados, buscando identificar variáveis relevantes, tipos de características (categóricas e numéricas), possíveis inconsistências e dados faltantes.

A atividade é desenvolvida em ambiente interativo, como o Google Colab ou o Jupyter Notebook, permitindo que código, texto e gráficos coexistam em um mesmo espaço de análise. Essa abordagem não apenas facilita a compreensão dos dados, mas também promove uma prática integrada entre programação e raciocínio analítico.

Aplicação Python

A aplicação em Python desenvolvida nesta atividade tem como foco a exploração e análise de dados reais fornecidos por uma empresa de cartão de crédito. Ela foi construída dentro de um notebook Python 3 — usando o Google Colab — e tem como objetivo aplicar conceitos fundamentais da ciência de dados para responder a um problema de negócio concreto.
